This is the Game Library Project. The purpose of this project is to add 
new games into your library and give details, such as: Name, Genre, 
Rating, Platform Type, GameMode Type, Recommended Age, and Difficulty 
Level. You will need to use the Main class to add, remove, list games
and you can also search games by Genre.

USER INSTRUCTIONS: 
To use the library 
You'll have to right click on the main class and 5 options will be displayed:
1.Add a game
2.Remove a game
3.View game list
4.Search by game genre
5.Exit

Add a game
    you'll be able to add a game to your library and add details such as Name, 
    Genre, Rating, Version and Platform.

Remove a game
    Depending on your game position, you write the position of the game that you want to remove.
    
View game list
    Will show all the games you've added to your library and their positions and will display a 
    couple of their details.
    
Search by game genre
    You will be given genre choices to select which game genre you want the system to display.

Exit
    Once you are done using the library, you can use the exit option to leave the library and be 
    done with what you're doing.
